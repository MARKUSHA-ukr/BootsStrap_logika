# 📦 Bootstrap_logika Project

## 📖 Опис

Цей проєкт показує базове використання Bootstrap для створення простого сайту.

## 📂 Що є в проєкті

1. Скачайте або склонуйте репозиторій
2. Відкрийте файл `index.html` у браузері
